# DESCON設定ツール - 他のPCでの実行方法

## 📋 前提条件

このアプリケーションを他のPCで実行するには、以下のソフトウェアが必要です：

### 必須ソフトウェア

1. **Node.js (LTS版)**
   - ダウンロード: https://nodejs.org/
   - インストール時に「Add to PATH」にチェックを入れる
   - 推奨バージョン: 18.x 以上

2. **Python 3.7以上**
   - ダウンロード: https://www.python.org/downloads/
   - インストール時に「Add Python to PATH」にチェックを入れる
   - 推奨バージョン: 3.8 以上

## 🚀 初回セットアップ手順

### 1. ファイルのコピー

1. このフォルダ全体を他のPCにコピー
2. フォルダ名に日本語が含まれている場合は、英数字のみの名前に変更することを推奨

### 2. 初回セットアップの実行

1. `セットアップ-初回実行.bat` をダブルクリック
2. 必要なソフトウェアがインストールされているか確認
3. 依存関係のインストールが自動で実行される
4. 完了まで待つ（5-10分程度）

### 3. アプリケーションの起動

1. `DESCON設定ツール.bat` をダブルクリック
2. 初回起動時は環境構築が自動で行われる（2-3分）
3. アプリケーションウィンドウが開く

## 🔧 トラブルシューティング

### Node.jsが見つからない

```
Node.jsが見つかりません
```

**解決方法:**
1. https://nodejs.org/ にアクセス
2. 「LTS」版をダウンロード
3. インストール時に「Add to PATH」オプションにチェック
4. インストール後、PCを再起動
5. 再度ツールを起動

### Pythonが見つからない

```
Pythonが見つかりません
```

**解決方法:**
1. https://www.python.org/downloads/ にアクセス
2. 最新のPython 3をダウンロード
3. インストール時に「Add Python to PATH」にチェック
4. インストール後、PCを再起動
5. 再度ツールを起動

### ポート使用エラー

```
ポート3002が使用中です
ポート8765が使用中です
```

**解決方法:**
1. 他のアプリケーションを終了
2. タスクマネージャーで該当ポートを使用しているプロセスを終了
3. 再度ツールを起動

### アプリケーションが起動しない

**解決方法:**
1. ウイルス対策ソフトが実行をブロックしていないか確認
2. 管理者として実行してみる
3. `python`フォルダを削除して再起動（環境が再構築されます）
4. `node_modules`フォルダを削除して`npm install`を実行

## 📁 ファイル構成

```
DESCON設定ツール/
├── DESCON設定ツール.bat          ← メインの起動ファイル
├── DESCON設定ツール-汎用版.bat    ← より堅牢な起動ファイル
├── セットアップ-初回実行.bat      ← 初回セットアップ用
├── README-他のPCでの実行方法.md   ← このファイル
├── app/                          ← フロントエンドアプリ
├── backend/                      ← Pythonバックエンド
├── scripts/                      ← 起動スクリプト
├── node_modules/                 ← Node.jsパッケージ
├── python/                       ← Python環境（初回起動時に自動作成）
└── public/                       ← 設定ファイル（YAML）
```

## 💡 推奨事項

1. **フォルダ名**: 日本語を含まない英数字のフォルダ名を使用
2. **管理者権限**: 初回実行時は管理者として実行することを推奨
3. **ネットワーク**: 初回起動時はインターネット接続が必要
4. **ディスク容量**: 最低2GB以上の空き容量を確保

## 🔄 更新方法

アプリケーションを更新する場合：

1. 新しいバージョンのファイルで上書き
2. `python`フォルダを削除
3. `node_modules`フォルダを削除
4. `セットアップ-初回実行.bat`を実行
5. `DESCON設定ツール.bat`で起動

# DESCON設定ツール - 自動インストール版

## 🚀 **簡単な使用方法**

### **最も簡単な方法（推奨）**

1. **`DESCON-Easy-Setup.bat`** を右クリック → **「管理者として実行」**
2. 自動で必要なソフトウェアがインストールされます
3. インストール完了後、自動でアプリケーションが起動します

### **手動インストールが必要な場合**

1. **`DESCON-Complete-Setup.bat`** を右クリック → **「管理者として実行」**
2. 複数の方法で自動インストールを試行します
3. 失敗した場合は手動インストールの手順が表示されます

## 📋 **自動インストールされるソフトウェア**

- **Node.js LTS版** (最新の安定版)
- **Python 3.11** (最新版)
- **npm** (Node.jsと一緒にインストール)

## 🔧 **インストール方法の優先順位**

### **DESCON-Easy-Setup.bat**
1. **Chocolatey** (Windows用パッケージマネージャー)
2. **winget** (Windows 10/11標準のパッケージマネージャー)
3. **手動ダウンロード** (上記が利用できない場合)

### **DESCON-Complete-Setup.bat**
1. **winget** (Windows 10/11標準)
2. **手動ダウンロード** (PowerShell使用)
3. **手動インストール手順の表示**

## 💻 **システム要件**

- **OS**: Windows 10/11 (64bit)
- **権限**: 管理者権限が必要
- **ネットワーク**: インターネット接続が必要（初回のみ）
- **ディスク容量**: 約2GB以上の空き容量

## 🎯 **推奨する起動ファイル**

### **初回使用時**
1. **`DESCON-Easy-Setup.bat`** - 最も簡単で確実
2. **`DESCON-Complete-Setup.bat`** - より詳細な診断機能付き

### **2回目以降**
1. **`Start-DESCON.bat`** - 最もシンプル
2. **`DESCON設定ツール-汎用版.bat`** - 診断機能付き

## 🔍 **トラブルシューティング**

### **「管理者として実行」が必要**
```
This script requires administrator privileges.
```
**解決方法**: ファイルを右クリック → 「管理者として実行」を選択

### **ネットワークエラー**
```
Failed to download installer
```
**解決方法**: 
1. インターネット接続を確認
2. ファイアウォール設定を確認
3. 手動インストールを実行

### **インストール失敗**
```
Installation failed
```
**解決方法**:
1. ウイルス対策ソフトを一時的に無効化
2. 管理者権限でコマンドプロンプトを開く
3. 手動でソフトウェアをインストール

### **手動インストール手順**

#### **Node.js**
1. https://nodejs.org/ にアクセス
2. 「LTS」版をダウンロード
3. インストーラーを実行
4. 「Add to PATH」にチェックを入れる
5. インストール完了後、PCを再起動

#### **Python**
1. https://www.python.org/downloads/ にアクセス
2. 最新のPython 3をダウンロード
3. インストーラーを実行
4. 「Add Python to PATH」にチェックを入れる
5. インストール完了後、PCを再起動
